Molekyle som magneter

Ladningsforskydning mellem to atomer, med forskellig ladning, der sidder sammen i en molekyle-forbindelse.

Elektronegativitet

Elektronegativitetsforskel

polær og upolær

blandbarhedsregler

  1. Molekyler med polære bindinger kan blandes med molekyler med polære bindinger (hvis de har dipol)
  2. Molekyler med upolære bindinger kan blandes med molekyler med upolære bindinger (og dipoler)
  3. Molekyler med polære bindinger kan ikke blandes med molekyler med upolære bindinger
  4. Molekyler der indeholder både polære og upolære bindinger kan typisk blandes delvist molekyler med polære og upolære bindinger
  5. Ionforbindelser kan opløses i polære væsker
